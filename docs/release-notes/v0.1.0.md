# v0.1.0

Initial public release of Codex Knowledge Workflow Kit.

## Highlights

* Local-first persistent memory layer for AI agents using an Obsidian-style Markdown vault
* Public demo vault with inbox, project, research, wiki, resources, content, plan, and system memory areas
* Durable memory structure under `99_系统/记忆/`
* Active project continuity with `_state/CURRENT.md`, `_state/DECISIONS.md`, and `_state/HANDOFF.md`
* Prompt classification, session-start context, memory injection, and Markdown validation scripts
* One-command vault initializer with tool adapters
* Adapters for Codex, Claude Code, Gemini CLI, and Cursor
* Agent Skills for daily signals, inbox triage, research, llm-wiki, knowledge parsing, archive, and content workflows
* GitHub Actions checks for layout, Python, Node, shell, JSON, initializer smoke tests, and public safety scanning

## Quick Start

```bash
git clone https://github.com/z-zkagami/codex-knowledge-workflow-kit.git
cd codex-knowledge-workflow-kit
node scripts/init-kit.mjs --vault "$HOME/path/to/my-vault" --tool codex
```

Install every bundled adapter:

```bash
node scripts/init-kit.mjs --vault "$HOME/path/to/my-vault" --tool all
```
