# WeChat Draft Writer

Use this skill when the user asks for a full article draft, rewrite, or section expansion.

## Workflow

1. Read the topic card, outline, source material, and style profile.
2. Confirm the target reader, thesis, evidence, and article archetype.
3. Draft with concrete scenes and source-backed claims.
4. Keep uncertain claims marked for fact checking.
5. Run the quality checklist before returning the draft.

## Output

- Draft article
- Source gaps
- Fact-check items
- Suggested title direction
- Next production step
