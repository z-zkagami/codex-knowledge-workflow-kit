# Style 14D Framework

Use these dimensions to profile writing style:

1. Opening pattern
2. Sentence rhythm
3. Paragraph length
4. Evidence density
5. Scene specificity
6. Humor level
7. Technical depth
8. Reader address
9. Emotional temperature
10. Metaphor use
11. Transition style
12. Ending pattern
13. Claim confidence
14. CTA style

For each dimension, record observed patterns, strong examples, and avoid rules.
