# WeChat Material Intake

Use this skill when raw material should become content planning input.

## Workflow

1. Read the source material and identify the target audience.
2. Extract facts, scenes, examples, claims, useful quotes, and unresolved questions.
3. Mark risk areas that need fact checking.
4. Suggest possible topic angles for `wechat-topic-outline-planner`.
5. Save or update a topic card under `60_Content/00_Topic_Pool/` when requested.

## Output

- Material summary
- Strongest usable examples
- Possible article angles
- Fact-check risks
- Recommended next skill
