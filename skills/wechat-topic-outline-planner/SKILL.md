# WeChat Topic Outline Planner

Use this skill when the user needs article topic selection, angle ranking, or outline planning.

## Workflow

1. Read source material, existing topic card, and style constraints.
2. Generate 3 to 5 candidate angles.
3. Score each angle for reader relevance, evidence, novelty, writeability, and cold-start clarity.
4. Select one recommended angle.
5. Produce an outline with section purpose, evidence needed, and narrative function.

## Output

- Candidate angles with scores
- Recommended angle
- Article archetype
- Draft outline
- Risks and missing sources
