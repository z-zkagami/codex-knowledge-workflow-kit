# Gemini CLI Adapter

This adapter installs `GEMINI.md` and `.gemini/persistent-memory.md`.

Run:

```bash
export CKW_KIT_ROOT="/path/to/codex-knowledge-workflow-kit"
export CKW_VAULT_ROOT="/path/to/your-vault"
node "$CKW_KIT_ROOT/scripts/init-kit.mjs" --vault "$CKW_VAULT_ROOT" --tool gemini-cli
```
