# Notice

This repository is Kagami's customized distribution of an AI-native knowledge workflow system.

## Origins

The vault structure started from the OrbitOS framework:

https://github.com/MarsWang42/OrbitOS

The public version in this repository focuses on Codex-native workflows, local-first knowledge operations, reusable Agent Skills, QMD semantic search, and content pipeline maintenance.

## Added Workflow Assets

Kagami's distribution adds or customizes:

* Codex/Agent operating rules in `AGENTS.md`
* Daily signal intake, inbox triage, and long-running `llm-wiki` workflows
* WeChat/article production skills
* Local script checks for vault structure, RSS capture, Flomo sync, and QMD embedding
* Reusable state, project, wiki, inbox, and writing templates
