# WeChat Style Profiler

Use this skill when the user provides article samples and asks to extract a reusable style profile.

## Workflow

1. Read representative samples.
2. Identify opening patterns, sentence rhythm, evidence use, section transitions, humor, tone, and ending patterns.
3. Separate stable style rules from one-off topic choices.
4. Produce a concise style DNA profile.
5. Store the profile under `50_Resources/Style_References/` when requested.

## Output

- Style DNA
- Do rules
- Avoid rules
- Reusable phrases or patterns
- Calibration checklist
