# Outline Patterns

## Pattern A 问题拆解型
- 开头：真实困惑或反常识观察
- 主体 1：问题为什么会发生
- 主体 2：常见误判或错误做法
- 主体 3：更有效的理解框架
- 主体 4：具体行动建议
- 结尾：收束与下一步

## Pattern B 案例推导型
- 开头：一个人或一件事的切片
- 主体 1：表层现象
- 主体 2：底层原因
- 主体 3：可复制的方法
- 结尾：适用边界与提醒

## Pattern C 趋势判断型
- 开头：趋势变化或信号
- 主体 1：旧方法为什么失效
- 主体 2：新变量在哪里
- 主体 3：个人或企业该如何调整
- 结尾：时间窗口与行动建议

## 每节都要补齐
- `Section Objective`
- `Core Claim`
- `Evidence Needed`
- `Transition To Next`
