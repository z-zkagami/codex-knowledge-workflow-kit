# Paper Summary

Use this skill when the user provides a paper, PDF, abstract, or research link and asks for analysis or summarization.

## Workflow

1. Extract title, authors, venue, date, and source URL or file path.
2. Summarize the research question, method, dataset, results, and limitations.
3. Identify what can be reused in agent workflows, memory systems, or content.
4. Mark weak evidence, missing baselines, or unclear assumptions.
5. Save the result as a structured Markdown note when requested.

## Output

- Paper metadata
- Core idea
- Method
- Evidence
- Limitations
- Practical relevance
- Follow-up questions
